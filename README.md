# Explore Agent

> An AI that explores the internet for you, instead of making you search.

**Explore.fun** is a futuristic landing page for an autonomous AI exploration agent.
Not a search engine — an agent that scans, filters, and returns structured insights.

---

## What it is

- Autonomous exploration — the agent decides what to read and what to skip
- Insight synthesis — patterns and signals, not summaries
- Multi-source intelligence — news, research, forums, reconciled
- Transparent reasoning — full exploration trace per conclusion

---

## Stack

- Vanilla HTML / CSS / JS — zero dependencies, opens in any browser
- [Spline](https://spline.design) — interactive 3D robot arm embed
- Google Fonts — Syne + Inter
- Canvas API — particle system with mouse interaction
- CSS animations — aurora background, glass morphism, scroll reveals

---

## Run locally

Just open `index.html` in a browser. No build step needed.

```bash
open index.html
```

Or serve with any static server:

```bash
npx serve .
```

---

## Deploy

Drop `index.html` onto [Vercel](https://vercel.com), [Netlify](https://netlify.com), or any static host.
